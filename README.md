# Arduino-Robotic-Arm-Control-System
Arduino-based robotic arm system using servo motors for multi-axis movement and basic industrial automation simulation.

##  Hardware Connections

### Servo Motor Connections

| Servo | Wire Color | Arduino Connection |
|-------|------------|--------------------|
| Base Servo | Red | 5V |
| Base Servo | Brown/Black | GND |
| Base Servo | Orange/Yellow | D9 |

| Arm Servo | Red | 5V |
| Arm Servo | Brown/Black | GND |
| Arm Servo | Orange/Yellow | D10 |

| Gripper Servo | Red | 5V |
| Gripper Servo | Brown/Black | GND |
| Gripper Servo | Orange/Yellow | D11 |

##  Power Distribution

- All **Red wires** → Connected to **5V**
- All **Brown/Black wires** → Connected to **GND**
- Signal wires → Connected to Arduino digital pins
